# Changelog - 2026-04-03

## Documentation Updates

### 1) Added a storage expansion overview guide

- Added Chinese guide: `docs/Storage_Expansion_Guide.md`
- Added English guide: `docs/en/Storage_Expansion_Guide.md`
- Explains beginner-facing concepts:
  - partitions
  - filesystems
  - mounting
  - `/overlay`
  - extroot
- Clearly separates:
  - data partition approach
  - extroot approach
  - reflash/rebuild approach

### 2) Added a beginner-friendly data partition tutorial

- The new guide now includes a complete walkthrough for creating a TF card data partition and mounting it at `/srv/storage`
- Coverage includes:
  - partition creation
  - ext4 formatting
  - manual mount verification
  - persistent `fstab` setup using `UUID`
  - reboot verification
  - recommended locations for Git repos, backup archives, and large files

### 3) Rewrote the Overlay / Extroot expansion guide

- Rewrote `docs/ExtendOverlaySize.md`
- Rewrote `docs/en/ExtendOverlaySize.md`
- Reframed extroot as an advanced option instead of the default recommendation
- Added critical warnings:
  - do not mount an empty partition directly to `/overlay`
  - migrate the existing overlay data first
  - prefer `UUID` over a fixed device name
  - do not disable ext4 journaling on always-on devices

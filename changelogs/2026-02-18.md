# 更新日志 - 2026年2月18日

## 前端更新

### 1) 首页默认语言自动识别

- 新增浏览器语言识别逻辑：
  - 访问根路径 `/` 时，优先读取本地已选语言偏好。
  - 若无本地偏好，则根据 `navigator.languages` 自动判断。
  - 非中文浏览器默认跳转到 `/en/`，中文浏览器保持中文首页。
- 语言切换按钮会写入本地偏好，避免用户反复被默认策略覆盖。

### 2) 首页内容层级优化（更充实但不过度浮夸）

- 新增项目 Badge 区（版本、设备、License、维护状态）。
- 增加核心能力要点说明，突出上手路径与场景价值。
- 新增“最近更新”区块，直接展示最新 changelog 条目。
- 新增首页指标卡片（文档数量、脚本数量、更新记录数量）。
- 移除“仅显示中文文档/English docs only”等弱价值文案。

### 3) Changelog 英文页链接修复

- 修复 Markdown 内部链接解析：
  - 处理 `changelogs/en/*.md` 这类路径时，自动映射到英文文档路由 `/en/m/...`。
  - 避免跳转到原始 `.md` 地址导致页面报错。

## 文档与日志同步

- 更新 `changelogs/README.md` 和 `changelogs/en/README.md` 索引。
- 明确 changelog 记录范围：`docs/`、`scripts/`、`frontend/`。

## 验证

- 前端构建通过（Next.js static export）。
- 中文与英文 changelog 页面可正常访问与互相切换。
